### 简答题答案

**1.** `list` 用于需要动态修改的有序集合，如 `items = []; items.append("new")`。`tuple` 用于不应被修改的固定数据，如坐标 `point = (10, 20)` 或函数返回多个值。

**2.** `==` 比较值是否相等，`is` 比较是否是同一个对象（内存地址）。

```python
a = [1, 2, 3]
b = [1, 2, 3]
a == b  # True，值相同
a is b  # False，不同对象
```

**3.** 因为默认参数在函数定义时只创建一次，可变对象会在多次调用间共享状态：

```python
def add_item(item, lst=[]):
    lst.append(item)
    return lst
print(add_item(1))  # [1]
print(add_item(2))  # [1, 2] —— 不是期望的 [2]！
```

应改为 `lst=None`，函数内 `if lst is None: lst = []`。

**4.** Python 推荐 EAFP（Easier to Ask Forgiveness than Permission），即优先用 `try/except`。因为：①代码更简洁，避免冗长的条件检查；②在多线程环境下，`if` 检查和实际操作之间状态可能改变；③异常路径性能开销仅在出错时才产生，正常路径更快。

**题目 1：**

```python
def format_table(rows, headers):
    col_widths = []
    for i, h in enumerate(headers):
        max_w = len(str(h))
        for row in rows:
            max_w = max(max_w, len(str(row[i])))
        col_widths.append(max_w)

    def fmt_row(values):
        return " | ".join(str(v).ljust(col_widths[i]) for i, v in enumerate(values))

    lines = [fmt_row(headers)]
    lines.append("-" * len(lines[0]))
    for row in rows:
        lines.append(fmt_row(row))
    return "\n".join(lines)
```

**题目 2：**

```python
# (1)
def deep_get(d, keys, default=None):
    current = d
    for key in keys:
        if isinstance(current, dict) and key in current:
            current = current[key]
        else:
            return default
    return current

# (2)
def flatten_dict(d, parent_key="", sep="."):
    items = {}
    for k, v in d.items():
        new_key = f"{parent_key}{sep}{k}" if parent_key else k
        if isinstance(v, dict):
            items.update(flatten_dict(v, new_key, sep=sep))
        else:
            items[new_key] = v
    return items
```

**题目 3：**

```python
import time

def retry(func, max_attempts=3, delay=0):
    last_exception = None
    for attempt in range(1, max_attempts + 1):
        try:
            return func()
        except Exception as e:
            last_exception = e
            if attempt < max_attempts:
                print(f"第 {attempt} 次重试...")
                if delay:
                    time.sleep(delay)
    raise last_exception
```

**题目 4：**

```python
def process_csv_data(lines, **transforms):
    headers = lines[0].split(",")
    result = []
    for i, line in enumerate(lines[1:], start=2):
        fields = line.split(",")
        if len(fields) != len(headers):
            print(f"警告：第 {i} 行字段数不匹配，已跳过")
            continue
        row = {}
        for h, v in zip(headers, fields):
            if h in transforms:
                row[h] = transforms[h](v.strip())
            else:
                row[h] = v.strip()
        result.append(row)
    return result
```
