### 题目 1：字符串处理

编写函数 `count_words(text)`，接收一个字符串，返回一个字典，统计每个单词出现的次数（忽略大小写）。

```python
# 示例
count_words("Hello hello world")
# 返回 {"hello": 2, "world": 1}

def count_words(text):
    wordDict = {}
    spliteData = text.split()
    print(spliteData)
    for word in spliteData:
        lowerV = word.lower()
        print(lowerV)
        if lowerV in wordDict:
            wordDict[lowerV] += 1
        else:
            wordDict[lowerV] = 1
    return wordDict
```

**要求：**
- 使用 dict 存储结果
- 忽略大小写
- 使用字符串的 split() 方法

### 题目 3：异常处理与 API 模拟

编写函数 `safe_divide(a, b)`，实现安全的除法运算：

```
def safe_divide(a, b):
    try:
        a = float(a)
        b = float(b)
        res = a / b

        print(f"{res:.2f}")
    except (ValueError,TypeError):
        print("错误：参数必须是数字")
    except ZeroDivisionError:
        print("错误：除数不能为零")
    finally:
        print("计算完成")
```



- 正常情况返回 `a / b` 的结果（保留两位小数）
- `b` 为 0 时返回 `"错误：除数不能为零"`
- 参数不是数字时返回 `"错误：参数必须是数字"`
- 无论成功失败，最后都打印 `"计算完成"`

### 题目 5：综合应用 - 简易数据过滤器

编写函数 `filter_data(data, **filters)`：
- `data` 是一个列表，元素为字典
- `**filters` 是任意数量的过滤条件
- 返回满足**所有**过滤条件的记录列表

```python
# 示例
products = [
    {"name": "手机", "price": 5000, "brand": "华为"},
    {"name": "电脑", "price": 8000, "brand": "联想"},
    {"name": "平板", "price": 3000, "brand": "华为"},
    {"name": "耳机", "price": 500, "brand": "小米"},
]

filter_data(products, brand="华为", price=5000)
# 返回 [{"name": "手机", "price": 5000, "brand": "华为"}]

def filter_data(data, **filters):
    # print(type(filters),filters["brand"],filters["price"])
    lst = []
    for prod in data:
        if filters["brand"] == prod["brand"] and filters["price"] == prod["price"]:
            dct = {
                "name":prod["name"],
                "price":prod["price"],
                "brand":prod["brand"]
            }
            lst.append(dct)
    print(lst)
```

**提示：** 遍历 filters 中的每个键值对，检查 data 中每条记录是否全部匹配。
