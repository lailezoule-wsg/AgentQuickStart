## 一、选择题（每题 2 分，共 20 分）

**1. 以下哪个是 Python 中不可变的数据类型？**

- A. `list`
- B. `dict`
- C. `set`
- D. `tuple`

**2. 下列代码的输出结果是？**

```python
x = "Hello"
y = "World"
print(f"{x} {y}!")
```

- A. `{x} {y}!`
- B. `Hello World!`
- C. `"Hello" "World"!`
- D. 报错

**3. `dict` 中访问一个不存在的 key 时，哪种方式不会报错？**

- A. `d["name"]`
- B. `d.get("name")`
- C. `d["name"] or None`
- D. `d.fetch("name")`

**4. 以下列表推导式的结果是？**

```python
[x**2 for x in range(5) if x % 2 == 0]
```

- A. `[0, 1, 4, 9, 16]`
- B. `[0, 4, 16]`
- C. `[4, 16]`
- D. `[1, 9]`

**5. 函数定义 `def foo(a, b=10, \*args, **kwargs)` 中，`**kwargs` 接收的是？**

- A. 位置参数组成的元组
- B. 关键字参数组成的字典
- C. 默认参数组成的列表
- D. 所有参数组成的集合

**6. 以下代码执行后，`result` 的值是？**

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    result = 0
finally:
    result = -1
```

- A. `0`
- B. `-1`
- C. `10`
- D. 报错

**7. `lambda x, y: x + y` 等价于哪个函数定义？**

- A. `def f(x): return x + y`
- B. `def f(x, y): return x + y`
- C. `def f(x, y): x + y`
- D. `def f(*args): return sum(args)`

**8. 以下哪种导入方式会产生命名空间污染？**

- A. `import math`
- B. `from math import sqrt`
- C. `from math import *`
- D. `import math as m`

**9. `set` 和 `list` 的主要区别是？**

- A. `set` 有序且可重复
- B. `set` 无序且不可重复
- C. `set` 支持索引访问
- D. `list` 不可变

**10. 以下代码的输出是？**

```python
nums = [1, 2, 3, 4, 5]
print(nums[1:4])
```

- A. `[1, 2, 3, 4]`
- B. `[2, 3, 4, 5]`
- C. `[2, 3, 4]`
- D. `[1, 2, 3]`

## 四、编程题（每题 10 分，共 50 分）

### 题目 1：字符串处理

编写一个函数 `count_words(text)`，接收一个字符串，返回其中每个单词出现的次数（用 `dict` 实现）。

```
输入: "hello world hello python world hello"
输出: {"hello": 3, "world": 2, "python": 1}
```

### 题目 2：列表推导式

用**一行列表推导式**完成以下需求：
给定一个列表 `nums = [1, -2, 3, -4, 5, -6, 7]`，筛选出所有正数并求其平方，返回结果列表。

```
期望输出: [1, 9, 25, 49]
```

### 题目 3：异常处理

编写一个函数 `safe_divide(a, b)`，实现安全除法：

- 正常情况返回 `a / b`
- 除数为 0 时返回 `"除数不能为0"`
- 参数类型错误时返回 `"参数必须是数字"`

### 题目 4：函数与 \*\*kwargs

编写一个函数 `build_profile(**kwargs)`，接收任意数量的关键字参数，将它们组装成一个用户信息字典并返回。要求：

- 如果缺少 `"name"` 键，设置默认值为 `"Unknown"`
- 如果缺少 `"age"` 键，设置默认值为 `0`

```
输入: build_profile(name="Alice", city="Beijing")
输出: {"name": "Alice", "age": 0, "city": "Beijing"}
```

### 题目 5：综合题

编写一个程序，模拟简单的 API 调用重试机制：

- 定义一个函数 `call_api(url, max_retries=3)`
- 模拟请求（用 `print` 代替），每次尝试打印 `"尝试请求 {url}，第 {n} 次"`
- 用 `try/except` 捕获异常，模拟随机失败（可用 `random.random() < 0.5` 判断）
- 成功时返回 `"请求成功"`，所有重试都失败时返回 `"请求失败，已重试 {max_retries} 次"`
- 使用 f-string 格式化所有输出

### 编程题参考答案

**题目 1：**

```python
def count_words(text):
    result = {}
    for word in text.split():
        result[word] = result.get(word, 0) + 1
    return result
```

**题目 2：**

```python
result = [x**2 for x in nums if x > 0]
```

**题目 3：**

```python
def safe_divide(a, b):
    try:
        return a / b
    except ZeroDivisionError:
        return "除数不能为0"
    except TypeError:
        return "参数必须是数字"
```

**题目 4：**

```python
def build_profile(**kwargs):
    kwargs.setdefault("name", "Unknown")
    kwargs.setdefault("age", 0)
    return kwargs
```

**题目 5：**

```python
import random

def call_api(url, max_retries=3):
    for i in range(1, max_retries + 1):
        print(f"尝试请求 {url}，第 {i} 次")
        try:
            if random.random() < 0.5:
                raise ConnectionError("请求失败")
            return "请求成功"
        except ConnectionError:
            if i == max_retries:
                return f"请求失败，已重试 {max_retries} 次"
```
